Table APP_DB

Database (ADB): Stores metadata for each Oracle environment used in developing or delivering the application(s).
Columns
Name Type Optional Default Comments
DB_ID NUMBER(38)     Database ID: Surrogate key for this table.
DB_NM VARCHAR2(80)     Database Name: The database SID, alias or Service Name where an instance of the application can be found.
DB_DESCR VARCHAR2(2000)     Database Description: Description of the database environment for the current record.
DB_ALIAS VARCHAR2(80) Y   Database Alias: Optional alias for the database, e.g. MSSDV1.
