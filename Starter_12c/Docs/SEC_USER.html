Table SEC_USER

User (SU): Basic table of users, their identifiers and primary contact methods. Users can include human beings, named automated agents, and even IDs for hardware.
Columns
Name Type Optional Default Comments
USER_ID NUMBER(38)     User ID: Surrogate key for this table, filled by SEC_USER_SEQ.
USER_NM VARCHAR2(320)     User Name: The user name, often called the User ID, which the user enters in a login page to identify themselves.
PREF_NM VARCHAR2(1020) Y   Preferred Name: The user's preferred name.
PMY_EMAIL_ADDR VARCHAR2(320) Y   Primary Email Address: The primary email address to use when contacting the user.
WORK_PHONE VARCHAR2(160) Y   Work Phone: The daytime or work telephone number to use when contacting the user.
