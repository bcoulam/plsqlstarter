Table APP_CHG_LOG_DTL

Change Log (ACLGD): Attributive entity. Tracks changes to individual columns for a given record in an application over time. The CHG_LOG_ID column links it back to the parent change transaction record which has all the metadata surrounding the captured change.
Columns
Name Type Optional Default Comments
CHG_LOG_ID NUMBER     Change Log ID: Identifying foreign key to APP_CHG_LOG. Combined with the column_nm, forms the unique key for each change detail item.
COLUMN_NM VARCHAR2(120) Y   Column Name: Name of the column where the change was detected.
OLD_VAL VARCHAR2(4000) Y   Old Value: The old value of the column. NULL is expected for Insert changes. Convert dates, numbers and monetary amounts with a full format before writing them here.
NEW_VAL VARCHAR2(4000) Y   New Value: The new value of the column. NULL is expected for Delete changes. Convert dates, numbers and monetary amounts with a full format before writing them here.
