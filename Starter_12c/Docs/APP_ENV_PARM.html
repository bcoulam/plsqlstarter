Table APP_ENV_PARM

Environment Parameter (AEVP): Stores the environment-specific parameters and configuration values used by the applications across the databases in an enterprise.
Columns
Name Type Optional Default Comments
ENV_ID NUMBER(38)     Environment ID: Foreign key to APP_ENV. The environment wich is configured by the parameter.
PARM_ID NUMBER(38)     Parameter ID: Foreign key to APP_PARM. The parameter for the environment of the current row.
PARM_VAL VARCHAR2(4000) Y   Parameter Value: The value of the parameter for a given environment.
HIDE_YN VARCHAR2(1)   'N' Hide Y/N: A flag that indicates if the parameter should be hidden from display (Y) or not (N).
