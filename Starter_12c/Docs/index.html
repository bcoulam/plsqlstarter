APP Applications (A): List of the applications or systems that wish to use the Core framework.
APP_CHG_LOG Change Log (ACLG): Tracks change transactions.
APP_CHG_LOG_DTL Change Log (ACLGD): Attributive entity.
APP_DB Database (ADB): Stores metadata for each Oracle environment used in developing or delivering the application(s).
APP_EMAIL Email (AEM): Generic table for use in applications that have email requirements.
APP_EMAIL_DOC Email Documents/Attachments (AEMD): Large textual or binary attachments sent out with emails recorded in APP_EMAIL.
APP_ENV Environment (AEV): Stores the unique environments across the enterprise.
APP_ENV_PARM Environment Parameter (AEVP): Stores the environment-specific parameters and configuration values used by the applications across the databases in an enterprise.
APP_LOCK Logical Locks (ALK): Stores logical locks for applications that have pessimistic locking requirements.
APP_LOG Logs (ALG): Application logging table.
APP_MSG Standard Messages (AM): Stores static messages and dynamic message templates used within the systems sharing Core.
APP_PARM Parameter (AP): Stores the parameters and configuration values used by the applications across the databases in an enterprise.
SEC_PMSN Permission (SP): List of permissions to actions or objects that can be assigned to roles.
SEC_ROLE Role (SR): Stores the list of valid roles for each application.
SEC_ROLE_PMSN Role Permission (SRP): Map of the permissions that apply to a given role.
SEC_USER User (SU): Basic table of users, their identifiers and primary contact methods.
SEC_USER_APP User Application (SUA): Map of the applications to which a user has been granted access.
SEC_USER_ROLE User Role (SUR): Map of roles granted to users.
